# Captain IL - Mission HOPE

A fundraising website for Captain IL's Mission HOPE initiative - a global movement for healing, unity, and Jewish resilience.

## About Captain IL

Captain IL is a global Zionist initiative dedicated to:
- Healing the Jewish people after the October 7 attack
- Reframing Israel's story for the world
- Building unity between Israel and the diaspora
- Supporting IDF veterans and families
- Creating post-traumatic growth programs

## Getting Started

### View the Website
Simply open `index.html` in any modern web browser.

### For Development
1. Clone or download this repository
2. Open `index.html` in your browser
3. Edit files as needed:
   - `index.html` - Content and structure
   - `styles.css` - Styling and design
   - `script.js` - Interactive features

### Customization

#### Colors
The color scheme is defined in CSS variables in `styles.css`:
```css
:root {
    --primary-blue: #0078D4;
    --light-blue: #4DA6FF;
    --dark-blue: #005A9E;
    --gold: #FFB900;
    /* ... */
}
```

#### Contact Information
Update the email addresses in the contact section (currently set to `Bbbaruchi@gmail.com`)

#### Donation Links
Add your donation platform links to the "Support Captain IL" buttons

## Deployment

### GitHub Pages
1. Push to GitHub repository
2. Go to Settings > Pages
3. Select main branch
4. Your site will be live at `https://[username].github.io/CAP_IL/`

### Netlify
1. Create account at netlify.com
2. Drag and drop the folder
3. Site goes live instantly with free HTTPS

### Custom Domain
Point your domain's DNS to your hosting provider (GitHub Pages, Netlify, etc.)
